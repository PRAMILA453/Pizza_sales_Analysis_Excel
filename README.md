# Pizza_sales_Analysis_Excel
To Analyze  the data and creating the dashboard using Excel
 🍕 Pizza Sales Dashboard (Excel)

## 📌 Overview
This project presents an interactive **Pizza Sales Analysis Dashboard (2015)** built in Microsoft Excel.  
It provides insights into sales performance by pizza size, category, product type, and monthly order trends.

## ✨ Features
- 📊 **Sales by Size**: Bar chart showing total sales across pizza sizes (L, M, S, XL, XXL).
- 🥧 **Category Breakdown**: Pie chart displaying sales distribution across Chicken, Classic, Supreme, and Veggie pizzas.
- 💰 **Total Sales KPI**: Highlighted metric showing overall sales (817,860.05).
- 📅 **Time Filters**: Month and quarter slicers for dynamic analysis.
- 🍕 **Top-Selling Pizzas**: Bar chart ranking pizzas such as Barbecue Chicken, Five Cheese, Hawaiian, Thai Chicken, etc.
- 📈 **Order Trends**: Line chart showing monthly order counts throughout the year.

## 🚀 How to Use
1. Open the Excel file (`Pizza_Sales_Dashboard.xlsx`).
2. Use slicers to filter by month or quarter.
3. Explore charts to analyze sales by category, product, and size.
4. Review KPIs and trends to identify performance insights.

## 🛠️ Tech Stack
- Microsoft Excel (Pivot Tables, Charts, Data Validation, Slicers)
- Dataset: Pizza Sales Data (2015)
